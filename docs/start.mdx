# 快速开始

:::tip

本文默认你已经了解并熟悉NodeJS编程，

如果你对NodeJS的部署和配置不熟悉，

同时也对JavaScript主流的打包工具不熟悉，

## 使用模板

- 获取模板

```sh
npm create alemonjs@latest -y
```

- 启动

```sh
yarn dev
```

## 手动构建

- 创建启动文件

```js title="index.js"
import { start } from 'alemonjs'
start({
  input: 'lib/index.js' // 模块入口文件
})
```

- 模块入口文件编写逻辑

```js title="lib/index.js"
import {
  defineChildren,
  defineRouter,
  logger,
  useMessage,
  Format
} from 'alemonjs'

const response = e => {
  // 使用消息工具
  const [message] = useMessage(e)
  // 构造消息内容
  const format = Format.create().addText('hello word')
  // 发送消息
  message.send({ format })
}

// 设置路由规则
const responseRouter = defineRouter([
  {
    regular: /^(\/|#)?hello$/,
    handler: () => response
  }
])

export default defineChildren({
  // 注册内容
  register() {
    return {
      responseRouter
    }
  },
  // 当注册完成时
  onCreated() {
    logger.info('本地测试启动')
  }
})
```

- 启动

```sh
node index.js
```
