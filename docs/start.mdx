## 使用模板

- 获取模板

```sh
npm create alemonjs@latest -y
```

- 启动

```sh
yarn dev
```

## 手动构建

- 定义本地模块入口

```json title="package.json"
{
  // 确保文件拥有以下内容
  "main": "lib/index.js", // 入口文件
  "type": "module" // esm
}
```

- 创建启动文件

```js title="index.js"
import { start } from 'alemonjs'
start()
```

- 模块入口文件编写逻辑

```js title="lib/index.js"
import {
  defineChildren,
  onSelects,
  defineResponse,
  logger,
  Text,
  useMessage
} from 'alemonjs'

const selects = onSelects(['message.create'])

const res = onResponse(selects, event => {
  // 创建
  const [message] = useMessage(event)
  // 发送消息
  message.send(format(Text('hello Word')))
})

// 设置路由规则
const response = defineResponse([
  {
    regular: /^(\/|#)?hello$/,
    handler: async () => res
  }
])

export default defineChildren({
  // 注册内容
  register() {
    return {
      response
    }
  },
  // 当注册完成时
  onCreated() {
    logger.info('本地测试启动')
  }
})
```

- 启动

```sh
node index.js
```
