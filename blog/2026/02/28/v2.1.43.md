# 更新记录

- 增强 v8 解析兼容性

- 默认无端口模式

- 修复若干兼容性BUG

- 主动消息

> 部分平台可能仅支持有限或不支持主动消息机制

```ts
import { MessageDirect, Format } from 'alemonjs'

const messageDirect = MessageDirect.create()

const format = Format.create()
format.addText('hello word')

// 向指定频道发送消息 。SpaceId 从消息中获得，注意这可能不是 ChannelId
messageDirect.sendToChannel({
  SpaceId: 'ChannelID',
  format
})

// 向指定用户发送消息  OpenID 从消息中获得，注意这可能不是 UserId
messageDirect.sendToUser({
  OpenId: 'UserID',
  format
})
```
